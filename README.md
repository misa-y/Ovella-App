# Ovella-App
Ovella's App concept. Symptom tracker with advocacy report generator. Wearable concept settings can be toggled on/off. 

## Features

- Daily pain check-in form
- Pain level tracking from 1 to 10
- Symptom, mood, energy, medication, cycle day, and notes logging
- Local demo data
- Visible data dashboard
- Weekly pain chart
- Advocacy report generator
- Basic settings screen
- Local browser storage using `localStorage`
